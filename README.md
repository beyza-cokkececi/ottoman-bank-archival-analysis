# Ottoman Bank Archival Analysis

This repository accompanies a research project on labor structures, wage dynamics, and personnel mobility within the Imperial Ottoman Bank.

It provides a reproducible workflow for transforming archival records into structured, machine-readable data using prompt-based extraction methods.

---

## 📌 Overview

The project combines:
- Archival personnel records
- Prompt engineering for information extraction
- Structured JSON outputs for quantitative analysis

The goal is to bridge qualitative historical sources with computational analysis.

---

## 📂 Repository Structure

- `documents_and_outputs/` → Sample archival records Example structured JSON outputs with GPT4o-mini and GPT-5 
- `prompts/` → Prompt templates used for extraction
